TIKET.com X BNCC

Tiket.com x BNCC is a collaboration program between Tiket.com, one of the leading technology companies in Indonesia, and BINA NUSANTARA Computer Club (BNCC), a computer organization at Bina Nusantara University.


This program aims to provide learning opportunities for BNCC members in various technology fields, such as web application development, mobile applications, and UI/UX. Through this program, BNCC members can access online courses, bootcamps, workshops and guidance from Tiket.com experts. They can also work on real projects and show their abilities and creativity.


One example of a project created by BNCC members with Tiket.com is BNCC x Tiket Movies, a web application that uses The Movie Database API to search for the latest films. This application was developed using Next.js, a framework for web development with React. This application has a movie search feature, movie details, and a list of favorite movies. You can view the source code of this application on GitHub.


If you are interested in finding out more about Tiket.com x BNCC, you can visit their official website or read several articles from BNCC members participating in this program. I hope this helps you understand what Tiket.com x BNCC is and what they do.
